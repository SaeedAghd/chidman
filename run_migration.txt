📋 دستورات اجرای Migration در Liara:

🔸 روش ۱ - از طریق Liara Console (آسان‌تر):
════════════════════════════════════════════════
1. برو به: https://console.liara.ir/apps/chidmano
2. از منوی سمت چپ، روی "Console" کلیک کن
3. دستور زیر را وارد کن:

   python manage.py migrate

4. Enter را بزن و منتظر بمان تا تمام شود

---

🔸 روش ۲ - از طریق SSH به سرور:
════════════════════════════════════════════════
liara shell --app chidmano

سپس:
python manage.py migrate

---

🔸 بررسی وضعیت Migration ها:
════════════════════════════════════════════════
python manage.py showmigrations store_analysis

باید ببینی که همه migration ها علامت [X] دارند

⚠️ نکات مهم:
════════════════════════════════════════════════
- بعد از اجرای Migration، سرور را Restart نکن (خودکار Reload می‌شود)
- اگر خطا دیدی، Screenshot بگیر و ارسال کن
- Migration ها خودکار و ایمن هستند (Rollback ندارند، پس قبلش Backup بگیر)
