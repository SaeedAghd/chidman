## 🚀 **راه‌اندازی:**

### **1. نصب Ollama:**
```bash
# Windows
winget install Ollama.Ollama

# Linux
curl -fsSL https://ollama.ai/install.sh | sh

# macOS
brew install ollama
```

### **2. راه‌اندازی سرویس:**
```bash
ollama serve
ollama pull llama3.2
```

### **3. اجرای Django:**
```bash
python manage.py migrate
python manage.py runserver
```

### **4. تست سیستم:**
```bash
python test_ollama_system.py
```
